# Logo Instructions

To complete the setup, please add the following logos to this folder:

## Required Logos:

1. **rice-bioengineering-logo.png** (or .jpg)
   - Rice University Bioengineering Department logo
   - Can be found at: https://bioengineering.rice.edu/
   - Used for: Cancer Bioengineering Research project

2. **duke-chemistry-logo.png** (or .jpg)
   - Duke University Department of Chemistry logo
   - Can be found at: https://chemistry.duke.edu/
   - Used for: Chemical Biology Research project

## How to Add:

1. Download the logos from the respective university websites
2. Save them in this `/public/images/` folder
3. Name them exactly as shown above (or update the file paths in Projects.jsx if you use different names)
4. The logos will automatically display on your portfolio

## File Format:
- PNG or JPG format
- Recommended: PNG with transparent background for best appearance
- The images will be automatically styled to fit properly in the project cards
